File Documentation

app.py

Purpose: Main Python script for the Flask application, handling HTTP requests and integrating the machine learning model.

Contents:

  • Import statements for necessary modules (Flask, Keras, TensorFlow, etc.).
  • Definition of the Flask application (app).
  • Loading of the trained machine learning model (chest_xray_2.h5).
  • Route definitions for the home page (/), prediction endpoint (/predict), and medication retrieval endpoint (/medications).
  • Functions to render the home page (index()), handle predictions (predict()), and retrieve medications (get_medications()).

Usage: Run this script to start the Flask application and access it via a web browser.

medicines.py

Purpose: Contains the MedicineFilter class, responsible for filtering medications based on criteria from a CSV file.

Contents:

  • Import statement for the csv module.
  • Definition of the MedicineFilter class with methods to initialize the class and filter medications.

Usage: Import this class in app.py to filter medications based on certain conditions.

styles.css

Purpose: Cascading Style Sheets (CSS) file defining the visual appearance and layout of HTML elements.

Contents:

  • Styles for various HTML elements, including fonts, colors, padding, margins, and layout.

Usage: Linked in HTML files to style the content and layout of web pages.

index.html

Purpose: HTML template for the home page of the Pneumonia Detection application.

Contents:

  • Header section with the application title.
  • Main section containing file upload functionality, result display area, and medications container.
  • Footer section with copyright information.

Usage: Rendered by the Flask application to display the home page in the web browser.

documentation.html

Purpose: HTML template for the documentation page of the Pneumonia Detection application.

Contents:

  • Header section with the documentation title.
  • Main section containing various sections such as Introduction, Installation, Usage, Features, and Deployment.
  • Footer section with copyright information.

Usage: Rendered by the Flask application to display the documentation page in the web browser.
